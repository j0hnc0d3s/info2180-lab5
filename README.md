# World Database Lookup — Country & City Search

> A search directory for the countries of the world and their capital cities.

Built for **INFO2180 — Web Programming I** at the University of the West Indies, Mona.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (jQuery)
- **Backend:** PHP
- **Data:** MySQL (world database)

## Features
- Look up country information by name
- View list of cities within a selected country
- AJAX-driven results with no page reload

## Run Locally
```bash
php -S localhost:8000
```
